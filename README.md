# ViaCEP - Angular 4.X

This is the repository of the ViaCEP (Angular 4.X) application

### Frontend Installation

Install the dependencies and start the application.

###### angular
```sh
$ npm install -g @angular/cli
```

###### root
```sh
$ cd viacep
```

###### dependencies
```sh
$ npm install
```

###### run
```sh
$ ng serve
```

###### browser
```sh
http://localhost:4200
```
